App-level configuration

The title found in the center of the page header should be changed in src/main/webapp/scripts/app.js. Specifically, the line that reads 

    .constant('appTitle', 'Polaris Angular App')       <-- changes the title in the center of the page header
    .constant('roleUrl', '/npoadmin/role')             <-- relative url where the user's role is retrieved from
    .constant('attributeUrl', '/npoadmin/attributes')  <-- relative url where the application attributes are retrieved from
    .constant('appTabs', [
        {id: 'tab1', desc: 'Page Tab 1', url: 'url', isActive: 'active'},
        {id: 'tab2', desc: 'Page Tab 2', url: 'url', isActive: ''},
        {id: 'tab3', desc: 'Page Tab 3', url: 'url', isActive: ''},
        {id: 'tab4', desc: 'Page Tab 4', url: 'url', isActive: ''}
    ])
